# Encounter Writing Style — Sablewood Messengers (pp. 22-39)

This is the official quickstart adventure that inspired the campaign. Use it as the structural template for all written encounters, hooks, and adventure content.

## Core Structure Patterns

### Acts
Adventures divide into named Acts with clear narrative purpose:
- Short title (e.g. "Act One: The Merchant Cart")
- One-sentence summary of what happens
- Each act has a clear trigger to move to the next

### Read-Aloud Boxes
- Italicized text in a bordered box
- Written in second person, present tense ("*you see…*", "*from within, you feel…*")
- Sets the scene; ends with an open question inviting player action ("*What would you like to do?*")
- GM guidance appears *outside* the box, in regular text

### Scenario Branches ("If they...")
After a scene, list conditional outcomes as bold "If they..." lines:
> **If they search the merchant cart,** describe X.
> **If they attack the strixwolf,** have them make an attack roll at difficulty 10.
> **If they fail any rolls...,** it will...

### Action Roll Results
Structured under three headers:

**SUCCESS**
- "If they rolled X or higher, it is a success!"
- Sub-branches: **If the roll was with Hope** / **If the roll was with Fear**
- Include a sample GM line in italics for Fear consequences

**FAILURE**
- "If they rolled X or below, it is a failure."
- Sub-branches: **If the roll was with Hope** / **If the roll was with Fear**
- Fear failure = Gain a Fear AND mark Stress

**CRITICAL SUCCESS**
- Triggered when both dice show the same number
- Player marks a Hope AND may clear a Stress
- Describe a particularly favorable outcome

### Stat Blocks

```
Adversary Name (Reference / Standard / etc.)

Tier X [Role]
Weapon - Range - Xd+Y [phy/mag]     Thresholds: X/Y
ATK: +X                              HP: ○ ○ ○
Difficulty: X                        Stress: ○ ○

FEATURES
Feature Name - Type: Description of what it does.
```

- Use "Reference" for groups of identical adversaries; list individual HP/Stress below
- Actions require spending Fear; Reactions trigger conditionally; Passives are always on

### NPC Mini-Profiles (for social scenes)
```
Name - Heritage Race - (pronouns)

One paragraph: physical description + personality hook + what they want from PCs.
```

### Environment Features
```
Location Name
Tier: X    Type: [Combat/Exploration/Social]

FEATURES
Feature Name - Type: Spend a Fear to [effect].
```

### Battle Guides
After stat blocks, include a plain-language "Battle Guide":
- How adversary attacks work step by step
- When to use special features
- Tactical advice (who they prioritize, when to use Fear moves)

### GM Prep Sections
Boxed callouts labeled **GM PREP DURING REST** or similar:
- Mechanical reminders timed to lulls in action
- Instructions for countdown dice, environment setup, adversary placement

## Tone & Voice
- GM text is direct and confident, never tentative
- Read-aloud prose is atmospheric, sensory, wonder-driven
- NPC dialogue is distinct and characterful (e.g. the Arcanist: "Oh my, you're rather late aren't you.")
- Consequences always push the story forward — even failures create interesting complications, not dead ends
- Lean into collaborative worldbuilding: ask players questions to fill in details ("*What unique thing have you noticed about the trees?*")

## Pacing Conventions
- Allow roleplay between scenes before triggering the next event
- Combat ends cleanly: "The encounter ends when all adversaries are defeated."
- Short rests have a menu of 4 structured options (Tend Wounds / Clear Stress / Repair Armor / Prepare)
- Epilogues zoom out narratively, then plant a hook for the next adventure
