# Daggerheart Campaign Vault

A complete campaign management system for running Daggerheart by Darrington Press.

## Folder Structure

```
daggerheart/
├── campaign/
│   ├── campaign-overview.md      <- Master campaign document (start here)
│   └── session-zero.md           <- Safety tools, tone, player contributions
│
├── characters/
│   ├── player-characters/        <- One file per PC (use _template-pc.md)
│   └── npcs/                     <- One file per notable NPC (use _template-npc.md)
│
├── world/
│   ├── locations/                <- One file per location (use _template-location.md)
│   ├── factions/                 <- One file per faction (use _template-faction.md)
│   ├── lore/
│   │   └── world-primer.md       <- Living world bible
│   └── maps/                     <- Store map images or ASCII maps here
│
├── adventures/
│   ├── sessions/                 <- One file per session (use _template-session.md)
│   ├── encounters/               <- Reusable encounter files
│   └── hooks/
│       └── plot-hooks.md         <- All active and banked hooks
│
├── mechanics/
│   ├── reference/
│   │   └── daggerheart-quick-reference.md
│   └── house-rules/
│       └── house-rules.md
│
└── assets/
    └── inspiration/
        └── mood-and-tone.md
```
